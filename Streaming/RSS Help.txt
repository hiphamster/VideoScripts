Using an RSS bot in chat may not be such a bad idea to let viewers know what else
you've been up to. You could essentially use OBS Studio's text file reader to display
RSS output piped to a text file from a terminal application like "rsstail."

YouTube:
So here’s how to create an RSS feed from any YouTube channel.
Steps

    Go to the YouTube channel you want to track
    View the page’s source code
    Look for the following text: channel-external-id
    Get the value for that element (it’ll look something like UCK5qGhtF5XS3ngbqyQ7iBqw
    Replace that value into this URL:

https://www.youtube.com/feeds/videos.xml?channel_id=UCK5qGhtF5XS3ngbqyQ7iBqw
